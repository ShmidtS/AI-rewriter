# AI Книжный Переписчик

> Инструмент для переписывания больших текстов (книги, статьи) через локальный LLM прокси.
> Поддерживает GUI и веб-интерфейс, многоязычный UI, несколько пресетов промптов.

## Возможности

- **Умное разбиение на блоки** — по семантическим границам (абзацы, предложения)
- **Контекстное переписывание** — каждый блок видит соседние для плавных переходов
- **Адаптивная генерация** — температура автоматически регулируется при ошибках качества
- **Валидация качества** — проверка схожести, соотношения длины, разнообразия лексики
- **Возобновление работы** — сохранение прогресса, продолжение с последнего блока
- **GUI** — десктопное приложение на tkinter с тёмной темой
- **Веб-интерфейс** — Flask + SSE для отображения прогресса в реальном времени
- **Многоязычный интерфейс** — Русский / English / 中文
- **5 пресетов промптов** — Литературный, Академический, Упрощение, Творческий, Перевод
- **Интеграция с локальным прокси** — OpenAI-совместимый API (например, LLM-API-Key-Proxy)

## Требования

- Python 3.10+
- Локальный LLM прокси, работающий на `http://127.0.0.1:8000/v1` (OpenAI-совместимый)

## Установка

```bash
git clone https://github.com/ShmidtS/AI-rewriter.git
cd AI-rewriter
python -m venv .venv

# Windows:
.venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env с настройками вашего прокси
```

## Использование

### Десктопный GUI (по умолчанию)

```bash
python main.py
# или явно:
python main.py --gui
# или через модуль:
```


## Конфигурация

Создайте `.env` из `.env.example`:

```env
# URL прокси (OpenAI-совместимый)
OPENAI_BASE_URL=http://127.0.0.1:8000/v1

# Токен авторизации для прокси
AUTH_TOKEN=sk-admin74203

# Модель по умолчанию
MODEL=colin/gpt-5.4

# Размер контекстного окна
CONTEXT_WINDOW=1000000

# Максимальное количество токенов на блок
MAX_OUTPUT_TOKENS=32768

# Целевой размер блока (символы)
BLOCK_TARGET_CHARS=15000

# Пределы соотношения длины переписывания
MIN_REWRITE_LENGTH_RATIO=0.40
MAX_REWRITE_LENGTH_RATIO=1.60

# Язык интерфейса: ru / en / zh
UI_LANG=ru
```

### Профили подключения

Приложение поддерживает три профиля подключения:

| Профиль | Описание |
|---------|----------|
| `direct` | Прямое подключение к LLM API |
| `proxy` | Подключение через локальный прокси (рекомендуется) |
| `auto` | Автоматический выбор в зависимости от доступности |

## Решение проблем

### Частые проблемы

| Проблема | Решение |
|----------|---------|
| `ModuleNotFoundError: No module named 'core'` | Запускайте из корня проекта, не из подкаталогов |
| Connection refused to proxy | Убедитесь, что прокси работает на `http://127.0.0.1:8000` |
| Частые ошибки валидации качества | Настройте `MIN_REWRITE_LENGTH_RATIO` и `MAX_REWRITE_LENGTH_RATIO` в `.env` |
| Ошибки нехватки памяти | Уменьшите `BLOCK_TARGET_CHARS` или `CONTEXT_WINDOW` |
| GUI не запускается | Проверьте версию Python (требуется 3.10+), переустановите `sv-ttk` |
