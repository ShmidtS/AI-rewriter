## 概述

AI-rewriter 提供 5 个内置提示词预设，每个都针对特定文本类型和改写目标进行了优化。每个预设包括：

- **系统提示词** — 指导 LLM 如何改写的指令
- **本地化名称** — UI 中的显示名称（EN, RU, ZH, DE, FR, ES, JA, KO）
- **类别** — UI 组织的分组
- **标签** — 搜索和过滤的关键词
- **语言强制** — 严格的输出语言控制，在运行时注入

### 语言强制

每个预设都包含一个**严格的语言强制块**，在运行时基于所选输出语言注入。该块指示模型：

- **仅**以指定语言（RU, EN, ZH, DE, FR, ES, JA, KO）输出文本
- **永远**不使用其他语言
- 不要翻译成英语、中文、俄语或任何其他语言
- 即使输入文本是其他语言，输出也必须保持目标语言
- 输出中的每个词都必须是目标语言

这对**翻译**预设至关重要，因为源语言与目标语言不同。

## 可用预设

### 文学编辑 (`literary`)

**最适合：** 小说、长篇故事、短篇小说、文学散文

**风格：** 引人入胜的叙述、生动的描写、流畅的行文

**描述：** 将文本转化为精雕细琢的文学散文，同时保留原意和情感冲击力。非常适合希望提升散文风格的小说作家。

**特点：**
- 丰富、富有感染力的语言
- 强烈的叙述声音
- 生动的感官细节
- 场景间的流畅过渡
- 以角色为中心的散文

**使用示例：**
```
原文：她走进房间。很黑。她感到害怕。
改写：她跨过门槛，进入了一片仿佛能压迫眼睛的黑暗之中。
她的心跳加速，每一次跳动都是对周围寂静的小小反抗。
```

**何时使用：**
- 小说章节
- 短篇小说
- 文学随笔
- 创意非虚构

---

### 学术风格 (`academic`)

**最适合：** 研究论文、学术文章、学位论文、学术文本

**风格：** 正式、精确、学术、客观

**描述：** 按照学术写作标准重写文本，具有正确的结构、精确的术语和学术语气。在提高清晰度的同时保持客观性。

**特点：**
- 正式学术语体
- 精确术语
- 清晰的论证结构
- 客观语气
- 正确的引用整合

**使用示例：**
```
原文：实验进行得很好，我们发现了一些关于人们如何记忆
事物的有趣结果。
改写：实验方法显示出显著的有效性，在人类受试者的认知
保留机制方面产生了实质性发现。
```

**何时使用：**
- 研究论文
- 期刊文章
- 学位论文章节
- 学术提案
- 文献综述

---

### 简化 (`simplified`)

**最适合：** 技术文档、教育内容、公众传播

**风格：** 清晰、简单、易懂、直接

**描述：** 通过简化词汇、缩短句子和澄清概念，使复杂内容对更广泛的受众可理解，同时不失去准确性。

**特点：**
- 简单词汇
- 短小清晰的句子
- 主动语态
- 具体例子
- 逻辑流畅

**使用示例：**
```
原文：量子计算算法的实现需要对信息处理范式进行
根本性的重新概念化。
改写：量子计算机需要我们以新的方式思考计算。与
只能是 0 或 1 的常规比特不同，量子计算机使用可以
同时是两者的量子比特。
```

**何时使用：**
- 技术文档
- 教育材料
- 面向公众的内容
- 指导指南
- 解释性文章

---

### 创意增强 (`creative`)

**最适合：** 营销文案、创意写作、品牌故事

**风格：** 生动、富有感染力、动态、令人难忘

**描述：** 为文本增添创意风格和情感共鸣。非常适合使内容更具吸引力、令人难忘和有影响力。

**特点：**
- 动态、多样的句子结构
- 情感钩子
- 令人难忘的短语
- 感官语言
- 创意隐喻

**使用示例：**
```
原文：我们的咖啡很好。我们仔细烘焙。今天试试。
改写：每一杯都讲述着阳光普照的山脉和细心的双手的
故事。我们的烘焙大师唤醒在您舌尖上舞动的风味——
巧克力、柑橘、一丝焦糖。这不仅仅是咖啡。这是品味
非凡的邀请。
```

**何时使用：**
- 营销材料
- 品牌故事
- 产品描述
- 创意写作练习
- 社交媒体内容

---

### 翻译与改编 (`translation`)

**最适合：** 跨语言改写、本地化、文化适应

**风格：** 文化适应、自然、流畅

**描述：** 为目标语言受众翻译和改编文本，考虑文化细微差别、习语和本地表达。超越字面翻译。

**特点：**
- 文化适应
- 目标语言的自然表达
- 习语表达
- 本地参考
- 语气保留

**使用示例：**
```
原文（英语）："It's raining cats and dogs outside."
改编为中文："外面下着倾盆大雨。"
（自然的中国成语表达）

原文（英语）："Break a leg!"
改编为中文："祝你好运！"
（文化上适当的等效表达）
```

**何时使用：**
- 书籍翻译
- 内容本地化
- 跨文化交流
- 国际营销
- 字幕改编

## 自定义说明

除了预设，您还可以通过两个字段提供自定义说明：

### 风格字段

添加特定的风格要求：

**示例：**
- "使用英式英语拼写"
- "保持对话语气"
- "使用第一人称视角"
- "使用短段落以提高网页可读性"
- "在适当处加入幽默"

### 目标字段

指定改写目标：

**示例：**
- "缩短 20%，保留所有要点"
- "增加情感冲击力"
- "为 12 岁受众简化"
- "添加更多具体例子"
- "加强行动号召"

## 示例

### 示例 1：小说章节

**预设：** 文学编辑
**风格：** "哥特氛围，维多利亚背景"
**目标：** "增加开场场景的紧张感"

**改写前：**
> 房子很旧。多年没人住了。约翰走向门。

**改写后：**
> 庄园在淤青般的天空中若隐若现，它的窗户像空洞的眼睛，见证了几十年的衰败。一代人以来，没有灵魂跨过它的门槛——直到现在。约翰的脚步在碎石上嘎吱作响，每一步都让他更接近这座房子长久以来埋藏的秘密。

---

### 示例 2：研究摘要

**预设：** 学术风格
**风格：** "APA 格式，心理学期刊"
**目标：** "澄清方法论部分"

**改写前：**
> 我们做了一个 50 人的研究。我们给他们看图片并问问题。结果很有趣。

**改写后：**
> 本研究采用被试间实验设计，共 50 名参与者（每组 n=25）。参与者接受一系列视觉刺激，随后进行半结构化访谈。结果显示组间存在统计学显著差异（p < .05），表明参与者反应存在有意义的变异。

---

### 示例 3：产品描述

**预设：** 创意增强
**风格：** "奢侈品牌语调，精致"
**目标：** "强调独特性和工艺"

**改写前：**
> 这款手表是金制的。有皮带。走时准确。

**改写后：**
> 由 18K 金打造，配以手工缝制的意大利皮革，这款时计不仅仅是一件器具——它是一份传承。每只手表需要制表大师 200 小时的精心组装，他们的家族世代传承这门艺术。当您佩戴它，您在手腕上承载着一个世纪的卓越。

---

### 示例 4：技术教程

**预设：** 简化
**风格：** "面向非技术用户"
**目标：** "使其可操作，步骤清晰"

**改写前：**
> 必须修改配置文件以指定数据库连接字符串的适当参数，包括但不限于主机名、端口、身份验证凭据和数据库名称。

**改写后：**
> 要连接数据库，打开配置文件并添加以下信息：
> 1. **主机：** 服务器地址（如 `localhost` 或 `192.168.1.1`）
> 2. **端口：** PostgreSQL 通常为 `5432`
> 3. **用户名：** 您的数据库用户名
> 4. **密码：** 您的数据库密码
> 5. **数据库名：** 您给数据库起的名称

## 最佳结果提示

1. **匹配预设与内容类型** — 使用错误的预设可能产生意外结果
2. **适度使用自定义说明** — 让预设完成大部分工作
3. **目标要具体** — "缩短 30%"比"缩短"效果更好
4. **审查和迭代** — 初次结果可能需要完善
5. **将预设与风格结合** — 使用预设作为基础，添加风格进行自定义
